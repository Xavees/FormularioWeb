# 📋 Formulário FullStack com TypeScript + PHP + MySQL

Projeto de formulário completo desenvolvido utilizando:

- HTML5
- CSS3
- Bootstrap
- TypeScript
- PHP
- MySQL
- Fetch API

O sistema permite:

✔️ Cadastro de usuários  
✔️ Consulta de usuários por e-mail  
✔️ Integração entre Frontend e Backend  
✔️ Comunicação assíncrona utilizando Fetch API  
✔️ Exibição dinâmica dos dados em tabela  
✔️ Validação de formulário com TypeScript  
✔️ Integração com banco de dados MySQL  

---

# 🚀 Tecnologias utilizadas

## Frontend
- HTML5
- CSS3
- Bootstrap 5
- TypeScript

## Backend
- PHP

- PDO
